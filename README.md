## Privacy

This repository is intended to be public. Do not commit API keys, private keys,
mnemonics, passwords, verification codes, private contact details, internal
meeting links, unpublished business information, or other people's personal data.

## Structure

- `profile.md`: learner profile and learning constraints.
- `learning-plan.md`: weekly route and daily rhythm.
- `daily/`: daily notes and check-in drafts.
- `tasks/`: course tasks, exercises, and proof-of-work records.
- `experiments/`: small AI x Web3 prototypes and technical experiments.
- `handbook-feedback/`: feedback items for Handbook pages.
- `hackathon/`: project ideas, plans, demos, and submission material.
- `submissions/`: records of submitted check-ins, tasks, or links.
- `templates/`: reusable note templates.

## Current Focus

Build a practical AI x Web3 foundation for development and Hackathon projects:
first establish shared language, then produce small experiments that can become
project proof-of-work.
